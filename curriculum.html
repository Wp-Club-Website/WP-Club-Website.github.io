CURRICULUM

  • 경일고등학교 자율동아리 WP (Web Programming) 동아리에서는 웹 개발에 필요한 기본적인 지식들을 학습하기 위해 1학기에서는 컴퓨터 언어를, 2학기에서는 웹 사이트 제작을 하는 순서로 커리큘럼이 진행됩니다.

  • Learn To Code

    WP 동아리는 맨 처음 컴퓨터 프로그래밍에서 필수로 필요한 코딩 언어를 접하기 위해 가장 기본 언어들인 HTML, CSS, Javascript를 먼저 공부하고 나중에 Python을 codeacadmey.com 이라는 코딩 교육 웹사이트를 통해 1학기 동안 공부를 진행합니다. 각 언어마다 주어지는 코스와 작은 프로젝트를 수행하며 익히는 방식으로, 언제든지 학습 사이트에 접속하여 학습을 할 수 있을 뿐만 아니라, 자기 복습과 이미 코스를 수행한 다른 학생들에게도 자발적으로 질문하여 컴퓨터 언어를 학습합니다.

  • The Structure : HTML

    웹사이트를 제작하는데 가장 기본이자 중요한 언어 중 하나인 HTML (Hyper Text Markup Language)은 WP 동아리 회원들이 처음으로 접했던 언어입니다. 글의 크기, 색깔만을 바꿀 수 있을 뿐만 아니라 글에 하이퍼링크 추가와 이미지, 동영상 등 여러가지 컨텐츠와도 연결시켜주는 유용하면서 기본적인 언어임을 동아리 회원들은 학습하였습니다.

  • Design The Site : CSS

  • Create Website

    매주 수요일, 회의실에 앉아서 컴퓨터를 키고 동아리 회원들을 교육 웹사이트에 접속하여 자신이 공부했던 코딩을 계속 학습합니다. 칠판 메모를 통한 복습은 물론, 가끔씩 수다도 떨면서 웃음꽃을 피우기도 합니다. 그렇게 동아리 회원들은 서로 가까워지고 공식 웹사이트 개발을 통해 같이 협력하여 서로의 부족한 학습을 채워주고 도와줍니다. 동아리 회원들의 열정, 정과 협동심을 포트폴리오를 통해 확인해보세요.

  • Publish To The Web

    WP 동아리 활동을 통해서 동아리 회원들은 모두 컴퓨터 프로그래밍에 대한 첫 발을 내딛었으며, 이 웹사이트를 개발함으로써 자신이 앞으로 직접 여러가지 컨텐츠를 만드는데 좋은 역할을 할 개발자가 될 수 있을 밑거름이 될 것이라고 동아리 스스로 기대를 하고 있습니다.

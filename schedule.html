동아리 학습 일정

웹 개발에 필요한 기본적인 지식들을 학습하기 위해 1학기에 웹 프로그래밍 언어를 배우고, 2학기에서는 웹 사이트 제작을 하는 순서로 커리큘럼이 진행됩니다. 앞으로 코딩 학습 동아리를 운영하는 고교생에게 도움이 되길 바랍니다.

학습 방법

매주 수요일, 회의실에 앉아서 컴퓨터를 키고 동아리 회원들을 교육 웹사이트에 접속하여 자신이 공부했던 코딩을 계속 학습합니다. 칠판 메모를 통한 복습은 물론, 가끔씩 수다도 떨면서 웃음꽃을 피우기도 합니다. 이렇게 동아리 회원들은 서로 가까워지고 공식 웹사이트 개발을 통해 같이 협력하여 서로의 부족한 학습을 채워주고 도와줍니다.

학습 일지 다운로드 (PDF)

학습 일정

  • 2017년 4월 19일 - 동아리 첫 모임 및 웹 프로그래밍 학습 방법 계획 짜기
  • 2017년 5월 24일 - 웹 프로그래밍 학습 사이트 선정 웹 개발 기본 언어들인 HTML, CSS, Javascript를 먼저 공부하기 위해 codeacadmey.com 이라는 코딩 교육 웹사이트를 통해 학습 진행. 각 언어마다 주어지는 코스와 작은 프로젝트를 수행하며 익히는 방식으로, 언제든지 학습 사이트에 접속하여 학습을 할 수 있을 뿐만 아니라, 자기 복습과 이미 코스를 수행한 다른 학생들에게도 자발적으로 질문하여 컴퓨터 언어를 학습이 가능합니다.
  • 2017년 5월 31일 / 6월 7일 - The Structure : HTML 웹사이트를 제작하는데 가장 기본이자 중요한 언어 중 하나인 HTML (Hyper Text Markup Language)은 WP 동아리 회원들이 처음으로 접했던 언어입니다. 글의 크기, 색깔만을 바꿀 수 있을 뿐만 아니라 글에 하이퍼링크 추가와 이미지, 동영상 등 여러가지 컨텐츠와도 연결시켜주는 유용하면서 기본적인 언어임을 동아리 회원들은 학습하였습니다.
  • 2017년 6월 7일 / 7월 12일 - Design The Site : CSS웹사이트를 아름답게 꾸며주는 CSS 언어는 웹사이트를 더 고급진 디자인으로 꾸며줄 뿐만 아니라 사진이나 동영상, 텍스트를 보기 편한 구조로 재배치해주는 중요한 역할을 맡는 언어입니다.
  • 2017년 7월 19일 - 동아리 웹 사이트 기획 및 역할 분담1학기 마무리 모임으로 동아리 웹 사이트 기획 및 각자 역할을 분담하여, 각 웹 페이지 기획안 제작 방학 중 Github을 통해 웹 페이지를 제작하기로 하였습니다.
  • 2017년 8월 23일 - 동아리 웹 사이트 제작 방학 중 개인별로 익힌 HTML과 CSS의 지식을 바탕으로 직접 프로그래밍을 해 봄으로써 다 같이 웹사이트를 제작하며 프로그래밍을 체험할 수 있고 각각에 맡은 역할을 다함으로써 동아리 회원들이 책임감을 기를 수 있게 되었습니다.
  • 2017년 8월 30일 - 1차 웹 사이트 버전 출시1차 버전을 Github를 이용하여 업로드 하는 방법을 학습하고 웹사이트를 웹에 직접 게시함으로서 웹사이트를 모든 사람들이 보고 둘러 볼 수 있도록 제공하여 WP 동아리가 추구하고자 했던 목적을 달성하게 되었습니다.
  • 2017년 12월 30일 - 2차 웹 사이트 버전 출시
    기획안을 바꾸어 내용을 요약하여, 2차 웹사이트를 버전을 출시하였습니다.
  • 2018년 8월 14일 - 3차 웹 사이트 버전 출시
    디렉토리 구조, 로고, 디자인 변경 등 웹 사이트 구조와 코드 수정을 포함하여 3차 버전을 출시하였습니다.
